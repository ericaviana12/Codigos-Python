Introdução

Python é uma das linguagens de programação mais populares e amplamente utilizadas no mundo. Com sua sintaxe simples e legibilidade, é uma ótima opção para iniciantes que desejam aprender a programar. Neste artigo, vamos apresentar os melhores códigos Python para iniciantes, que ajudarão você a dar os primeiros passos nessa linguagem incrível.

1.	Olá, Mundo!

O “Olá, Mundo!” é o programa clássico de introdução em qualquer linguagem de programação. Ele consiste em exibir a frase “Olá, Mundo!” na tela. Em Python, o código para isso é extremamente simples:

    print("Olá, Mundo!")

2.	Calculadora Simples

Uma calculadora simples é um ótimo exercício para começar a explorar os recursos do Python. Com apenas algumas linhas de código, podemos criar uma calculadora que realiza operações básicas, como adição, subtração, multiplicação e divisão. Veja um exemplo:

    num1 = float(input("Digite o primeiro número: "))
num2 = float(input("Digite o segundo número: "))

soma = num1 + num2
subtracao = num1 - num2
multiplicacao = num1 * num2
divisao = num1 / num2

print("Soma:", soma)
print("Subtração:", subtracao)
print("Multiplicação:", multiplicacao)
print("Divisão:", divisao)

3.	Verificando se um Número é Par ou Ímpar

Outro exercício interessante é criar um programa que verifica se um número é par ou ímpar. Podemos utilizar o operador de módulo (%) para obter o resto da divisão por 2. Se o resto for igual a 0, o número é par; caso contrário, é ímpar. Veja o código:

    num = int(input("Digite um número: "))

if num % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

4.	Gerando Números Aleatórios

Python possui uma biblioteca chamada “random” que permite gerar números aleatórios. Podemos usar essa biblioteca para criar um programa que gera um número aleatório dentro de um intervalo especificado. Veja o código:

    import random

num_min = int(input("Digite o valor mínimo: "))
num_max = int(input("Digite o valor máximo: "))

numero_aleatorio = random.randint(num_min, num_max)

print("Número aleatório:", numero_aleatorio)

Aprenda a Programar em Python com Exemplos Práticos

Aprender a programar em Python pode parecer um desafio no início, mas com exemplos práticos, fica muito mais fácil compreender os conceitos e desenvolver suas habilidades. Nesta seção, vamos explorar alguns exemplos práticos que o ajudarão a aprender a programar em Python de forma divertida e interativa.

1.	Jogo da Velha

O jogo da velha é um clássico dos passatempos e pode ser implementado facilmente em Python. Com o uso de listas e estruturas de controle, podemos criar um programa que permite dois jogadores se enfrentarem em uma partida de jogo da velha. Veja o código:

tabuleiro = [" " for _ in range(9)]

def imprimir_tabuleiro():
    print("|" + tabuleiro[0] + "|" + tabuleiro[1] + "|" + tabuleiro[2] + "|")
    print("|" + tabuleiro[3] + "|" + tabuleiro[4] + "|" + tabuleiro[5] + "|")
    print("|" + tabuleiro[6] + "|" + tabuleiro[7] + "|" + tabuleiro[8] + "|")

imprimir_tabuleiro()

2.	Conversor de Temperatura

Um conversor de temperatura é um exemplo prático que demonstra como realizar cálculos simples em Python. Podemos criar um programa que converte temperaturas de Celsius para Fahrenheit e vice-versa. Veja o código:

    def celsius_para_fahrenheit(temp_celsius):
    return (temp_celsius * 9/5) + 32

def fahrenheit_para_celsius(temp_fahrenheit):
    return (temp_fahrenheit - 32) * 5/9

temp_celsius = float(input("Digite a temperatura em Celsius: "))
temp_fahrenheit = float(input("Digite a temperatura em Fahrenheit: "))

print(temp_celsius, "graus Celsius equivalem a", celsius_para_fahrenheit(temp_celsius), "graus Fahrenheit.")
print(temp_fahrenheit, "graus Fahrenheit equivalem a", fahrenheit_para_celsius(temp_fahrenheit), "graus Celsius.")

3.	Gerador de Senhas Aleatórias

A segurança é uma preocupação constante nos dias de hoje. Podemos criar um programa em Python que gera senhas aleatórias e seguras. Com o uso da biblioteca “random” e de strings, podemos criar senhas complexas que ajudarão a proteger suas informações. Veja o código:

    import random
import string

def gerar_senha(tamanho):
    caracteres = string.ascii_letters + string.digits + string.punctuation
    senha = ''.join(random.choice(caracteres) for _ in range(tamanho))
    return senha

tamanho_senha = int(input("Digite o tamanho da senha desejada: "))

print("Senha gerada:", gerar_senha(tamanho_senha))

4.	Contador de Palavras em um Texto

Um programa que conta a quantidade de palavras em um texto pode ser útil em diversas situações. Podemos criar um programa em Python que recebe um texto como entrada e retorna o número de palavras presentes nele. Veja o código:

    texto = input("Digite um texto: ")

palavras = texto.split()
quantidade_palavras = len(palavras)

print("O texto contém", quantidade_palavras, "palavras.")
