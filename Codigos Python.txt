183.	Verificando se um Número é Par ou Ímpar

19Outro exercício interessante é criar um programa que verifica se um número é par ou ímpar. Podemos utilizar o operador de módulo (%) para obter o resto da divisão por 2. Se o resto for igual a 0, o número é par; caso contrário, é ímpar. Veja o código:

22    print("O número é par.")

43Um conversor de temperatura é um exemplo prático que demonstra como realizar cálculos simples em Python. Podemos criar um programa que converte temperaturas de Celsius para Fahrenheit e vice-versa. Veja o código:

44    def celsius_para_fahrenheit(temp_celsius):
45    return (temp_celsius * 9/5) + 32

533.	Gerador de Senhas Aleatórias

54A segurança é uma preocupação constante nos dias de hoje. Podemos criar um programa em Python que gera senhas aleatórias e seguras. Com o uso da biblioteca “random” e de strings, podemos criar senhas complexas que ajudarão a proteger suas informações. Veja o código:

55    import random
56import string

57def gerar_senha(tamanho):
58    caracteres = string.ascii_letters + string.digits + string.punctuation
59    senha = ''.join(random.choice(caracteres) for _ in range(tamanho))
60    return senha

61tamanho_senha = int(input("Digite o tamanho da senha desejada: "))

62print("Senha gerada:", gerar_senha(tamanho_senha))
